λ Calculus Interpreter 
 BNF:
    exprs:  constant  
            | variable
            | "(" exprs exprs ")"
            | "(" "lambda" variable+ "." exprs ")"

    constant:
            number
            |  "(" "lambda" variable+ "." exprs ")"
    number: [0-9]+

    variable: [a-Z]
Examples:
  1. (lambda x. x) 1
  2. (lambda x. x 10) (lambda y.y)
  3. (lambda x. 5 x) (lambda y. 1)
  4. (lambda x. (lambda y.y) x) (lambda z. z)
