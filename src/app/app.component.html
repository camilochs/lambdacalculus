λ Calculus Interpreter
Información
Este intérprete fue desarrollado para el libro: Computación y programación funcional. Editorial Marcombo. Véase más información aquí.
 BNF:
    exprs:  constant  
            | variable
            | "(" exprs exprs ")"
            | "(" "lambda" variable "." exprs ")"

    constant:
            number
            |  "(" "lambda" variable "." exprs ")"
    number: [0-9]+

    variable: [a-Z] ([0-9])?
Ejemplos:
  1. (lambda x. x) 1
  2. (lambda x. x 10) (lambda y.y)
  3. (lambda x. (lambda y.y) x) (lambda z. z)
  4. (lambda f. ((lambda x2.(f (x2 x2))) (lambda x1.(f (x1 x1))))) (lambda x.x)
